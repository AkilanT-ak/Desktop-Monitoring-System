# Desktop Activity Monitoring & Screen Recording System (Academic Project)

## 📌 Overview
This project demonstrates a desktop monitoring system that records screen activity and generates periodic reports. 
The system supports:

- Screen recording for a fixed duration
- Periodic report generation
- Offline storage of reports
- Optional cloud/email sharing (for authorized personal use only)

⚠ This project is strictly for academic and personal research purposes.
It must not be used on devices without explicit user consent.

---

## 🛠 Technologies Used

- Python
- OpenCV
- PyAutoGUI
- SMTP (Email)
- Google Drive API
- Threading & Scheduling

---

## 🔄 System Workflow

1. Capture user activity
2. Record screen for fixed duration
3. Check internet connectivity
4. If online → Send report
5. If offline → Store locally
6. Retry when connection is available
